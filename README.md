## 🚀 TeamFlow Project

> **팀의 작업 관리, 회의 기록, 캘린더를 하나의 워크스페이스에서 관리하는 협업 툴**

---
### 🔗 프로젝트 배포 링크
https://teamflow-gold.vercel.app/

---

### 📌 프로젝트 개요

**TeamFlow**는 스터디 그룹, 프로젝트 팀 등을 위한 올인원 협업 플랫폼입니다.  
작업 보드, 회의록, 캘린더 기능을 통해 **일정 조율**과 **기록 확인**, **회의록 작성**을 지원합니다.

---

### 🗓 프로젝트 일정

#### 2025.06.11 ~ 2025.07.18

| 기간    | 내용                              |
| ------- | --------------------------------- |
| 1주차   | 기획, 디자인 시안, 기술 스택 선정 |
| 2~3주차 | 기능 구현 및 API 연동             |
| 4주차   | 마무리, 배포, 발표                |

---

### 💬 팀 협업 방식

- **GitHub Project 를 통한 대시보드 관리**
  - 실제 협업 툴을 사용하며 개발 플로우 및 기능 익힘에 도움
- **GitHub Issues + Branch 전략**
  - 이슈 번호 기반 브랜치 ex) `feature/#1-project-setting`
- **Commit Convention**
  - `feat:`, `fix:`, `docs:`, `refactor:`, `style:` 등
- **notion으로 팀 내부 문서관리**
  - [노션 링크](https://www.notion.so/1-2071254c9a5c802facc0f08aab172c85)
